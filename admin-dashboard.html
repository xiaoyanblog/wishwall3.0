记录 IP

开启后，新留言会保存访问者明文 IP，并可在留言详情里查看。

留言限制

按明文 IP 统计每日留言次数，适合防止刷屏。

验证码

可接入 Cloudflare Turnstile、hCaptcha、reCAPTCHA 等兼容 siteverify 的验证 API。

邮件通知

开启后，每当有人发布新留言，就会把留言内容发送到指定邮箱，并在下方收件箱记录发送结果。

邮箱信息

发件邮箱需要是 SMTP 服务或 Brevo 中已允许发信的邮箱；收件邮箱就是接收新留言通知的目标地址。

Brevo API

使用 Brevo Transactional Email API 发信。API Key 只会保存在服务端数据库，不会暴露给前台页面。

传统 SMTP

适合使用自己的邮箱服务商。587 通常使用 STARTTLS；465 通常开启 SSL/TLS。

通知收件箱

这里记录每次新留言通知和测试邮件的发送状态。

还没有通知记录
